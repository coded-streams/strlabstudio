      About FlinkSQL Studio
      
    

    

      

        

          
        

        

          
FlinkSQL Studio

          
Browser-based SQL IDE for Apache Flink

          
v1.0.0 · Apache 2.0 License

        

      

      

        FlinkSQL Studio was built to make Apache Flink SQL more accessible — a proper IDE experience without the overhead of a full platform deployment. Write, test, and debug Flink SQL statements from any browser, against any cluster.
